# Dependencies: 
- Headless cms by Nodejs + expressjs 
- Next.js + graphql
- Mysql

# Installation: 
- For backend : 
    - Config the database at database.js (create new schema if it not exist)
    - Run on the root directory:
<code> yarn </code>
<code> yarn build  </code>

- For frontend: 
<code> yarn </code>
<code> yarn dev  </code>

# How to build: 

- For backend : 
<code> yarn start  </code>

- For front end: 
<code> yarn dev  </code>
